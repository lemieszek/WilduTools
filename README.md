# WilduTools

A collection of tools for World of Warcraft.

## Features

- **Update Target Macro (UTM)**: Automatically update target macros
- **AutoGossip**: Automatically select gossip options when only one is available
- **UI Improvements**: Various UI enhancements and quality of life improvements

## Installation

1. Download the latest release
2. Extract the contents to your World of Warcraft `_retail_/Interface/AddOns` folder
3. Make sure the folder is named `!WilduTools`
4. Enable the addon in-game

## Usage

### Slash Commands

- `/wt` or `/wildutools` - Open the configuration panel
- `/utm` - Update Target Macro commands
- `/ccb` - Open Click Cast Binding
- `/kb` - Open Quick Keybind Mode

### Configuration

The addon can be configured through the in-game interface options panel. Type `/wt` to open it.

## Modules

### AutoGossip
Automatically selects gossip options when only one is available. Hold Shift to disable this behavior temporarily.

### UI Improvements
Various UI enhancements to improve WoW UI.
